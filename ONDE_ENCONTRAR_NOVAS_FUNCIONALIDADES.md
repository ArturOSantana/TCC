# 📍 Onde Encontrar as Novas Funcionalidades de Analytics

## 🎯 Localização Principal: Página de Insights

As novas funcionalidades de análise avançada estão na **página "Resumo & Análise"** do aplicativo.

### Como Acessar:

1. **Abra o app** e faça login
2. **Navegue até** a página de **Insights/Resumo & Análise**
3. **Selecione uma viagem** específica (não use "Visão Geral")
4. **Navegue pelas 4 abas** disponíveis

---

## 📊 As 4 Abas de Análise

### 1️⃣ **ABA RESUMO** (Dashboard)
**Ícone:** 📊 Dashboard

**O que você encontra:**
- ✅ **Índice de Eficiência**: Mostra se você está gastando mais rápido ou mais devagar que o planejado
- ✅ **Recomendações Inteligentes**: Sugestões baseadas nos seus padrões de gastos
- ✅ **Saúde Financeira**: Visão geral do orçamento vs gastos
- ✅ **Estatísticas Detalhadas**: Média, mediana, desvio padrão, quartis

**Exemplo de Recomendação:**
```
⚠️ Você está gastando 20% mais rápido que o planejado.
   Considere reduzir gastos em Alimentação.
```

---

### 2️⃣ **ABA GRÁFICOS** (Visualizações)
**Ícone:** 📈 Show Chart

**O que você encontra:**

#### 🎯 **Velocímetro de Orçamento** (Gauge Chart)
- Mostra visualmente quanto % do orçamento foi usado
- Cores indicam status:
  - 🟢 Verde (0-50%): Excelente
  - 🟠 Laranja (50-75%): Bom
  - 🔴 Vermelho (75-100%): Atenção/Crítico

#### 📈 **Gráfico de Linha Temporal**
- Evolução dos gastos dia a dia
- **Linha azul**: Gastos diários
- **Linha pontilhada**: Média móvel de 3 dias (suaviza variações)
- **Linha vermelha**: Orçamento diário planejado
- **Área preenchida**: Visualização do volume de gastos

#### 🌊 **Gráfico de Cascata** (Waterfall)
- Mostra como o orçamento foi consumido
- Começa com orçamento inicial
- Cada barra representa uma categoria de gasto
- Termina com saldo final
- **Verde**: Saldo positivo
- **Vermelho**: Saldo negativo

---

### 3️⃣ **ABA ANÁLISE** (Detalhamento)
**Ícone:** 📊 Analytics

**O que você encontra:**

#### 📊 **Distribuição por Categoria**
- Gráfico de barras horizontais
- Mostra quanto foi gasto em cada categoria
- Percentual do total

#### 📅 **Gastos por Dia da Semana**
- Identifica padrões: você gasta mais em qual dia?
- Útil para planejar futuras viagens

#### ⚠️ **Gastos Atípicos** (Outliers)
- Lista gastos muito acima ou abaixo da média
- Ajuda a identificar:
  - Eventos especiais (gastos altos)
  - Oportunidades de economia (gastos baixos)

**Exemplo:**
```
🔴 R$ 450,00 - Restaurante Gourmet
   (3.2x acima da média de Alimentação)
```

---

### 4️⃣ **ABA CALOR** (Heatmap)
**Ícone:** 📅 Calendar View Month

**O que você encontra:**

#### 🔥 **Mapa de Calor de Gastos**
- Calendário visual da viagem
- Cores indicam intensidade de gastos:
  - 🟦 Azul claro: Gastos baixos
  - 🟨 Amarelo: Gastos médios
  - 🟧 Laranja: Gastos altos
  - 🟥 Vermelho: Gastos muito altos
- **Passe o mouse/toque** em um dia para ver o valor exato
- Identifica padrões semanais e picos de gastos

---

## 🎓 Conceitos Importantes

### 📈 **Índice de Eficiência**
```
Eficiência = (% Orçamento Usado) / (% Tempo Decorrido)

- < 1.0: Você está economizando! 🎉
- = 1.0: No ritmo perfeito ✅
- > 1.0: Gastando mais rápido que o planejado ⚠️
```

### 📊 **Média Móvel de 3 Dias**
Suaviza variações diárias para mostrar a tendência real de gastos.

### 📉 **Outliers (Gastos Atípicos)**
Gastos que estão 1.5x acima ou abaixo do intervalo interquartil (IQR).

### 🔥 **Taxa de Queima (Burn Rate)**
Quanto você gasta por dia em média.

## 🚀 Como Usar Efetivamente

### Para Viagens em Andamento:
1. ✅ Verifique o **Velocímetro** diariamente
2. ✅ Leia as **Recomendações** na aba Resumo
3. ✅ Monitore o **Gráfico de Linha** para ver tendências
4. ✅ Ajuste gastos se o índice de eficiência > 1.2

### Para Análise Pós-Viagem:
1. ✅ Revise o **Mapa de Calor** para identificar padrões
2. ✅ Analise **Gastos Atípicos** para entender eventos especiais
3. ✅ Veja **Distribuição por Categoria** para planejar próximas viagens
4. ✅ Use **Gastos por Dia da Semana** para otimizar futuras viagens

## 🎨 Recursos Visuais

### Cores Adaptativas
Todos os gráficos se adaptam ao tema (claro/escuro) automaticamente!

### Interatividade
- **Toque** nos gráficos para ver detalhes
- **Arraste** para rolar gráficos grandes
- **Zoom** em gráficos de linha (se disponível)

## 📱 Requisitos

- ✅ Ter pelo menos **uma viagem criada**
- ✅ Ter **gastos registrados** na viagem
- ✅ Selecionar uma **viagem específica** (não "Visão Geral")

---

## 🐛 Solução de Problemas

### "Sem dados para exibir"
- Certifique-se de ter gastos registrados
- Verifique se selecionou uma viagem específica

### Gráficos não aparecem
- Recarregue a página
- Verifique sua conexão com internet
- Certifique-se de que a viagem tem datas válidas

### Cores estranhas no modo escuro
- Isso foi corrigido! Todos os gráficos agora usam cores adaptativas
